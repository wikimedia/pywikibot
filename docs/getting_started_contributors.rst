Getting Started for Contributors
================================

This page is intended for first-time contributors who want to get started
with Pywikibot development.

If you are new to Pywikibot or open source in general, this guide will help you
set up your environment, understand the project structure, and make your first
contribution.

Who this guide is for
---------------------

This guide is useful if you:

- Are new to Pywikibot
- Want to contribute code, documentation, or tests
- Are familiar with basic Git and Python concepts

Prerequisites
--------------

Before you begin, make sure you have:

- Python 3.8 or newer installed
- Git installed
- A GitHub account
- Basic familiarity with the command line

Project structure (high-level)
-------------------------------

Some important directories in the Pywikibot repository:

- ``pywikibot/`` – Core library code
- ``scripts/`` – User-facing bot scripts
- ``docs/`` – Documentation (this guide lives here)
- ``tests/`` – Test suite

Next steps
----------

After setting up your local environment, you can:

- Look for beginner-friendly issues
- Improve documentation
- Add tests
- Fix small bugs
